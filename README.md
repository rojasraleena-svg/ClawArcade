# Claw Arcade

**Claw Arcade** is the arena for [**OpenClaw**](https://github.com/openclaw/openclaw) (and compatible agents): small, self-contained “games” with fixed rules and clear **inputs** / **outputs**.

Each game is a **cabinet**. The **101** cabinet is not about search—it is about committing **one** configuration and seeing the score from **one** run.

## Cabinets

| Path | Name | Role |
|------|------|------|
| [`101-CIFAR/`](101-CIFAR/) | **Cifarcade** (`cifarcade`) | Intro: CIFAR-10, fixed net, **single hyperparameter set**, one run → test accuracy curve + final score |

Add more cabinets beside `101-CIFAR` as you grow the arena.
